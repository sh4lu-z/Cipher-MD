## 🚀 Setup & Deployment Guide

Follow these exact steps to get your CIPHER-MD bot up and running smoothly:

### 1️⃣ Fork the Repository
First, you need to fork this repository to your own GitHub account.
👉 **[Click Here to Fork](https://github.com/sh4lu-z/cipter-md/fork)**

### 2️⃣ Connect Bot & Get Session ID
Connect the bot to your WhatsApp account. Once successfully connected, you will receive your unique `SESSION_ID`.
👉 **[Connect Bot & Get Session ID](https://cipher-md.vercel.app/connect)**

### 3️⃣ Set Up Variables
Before deploying, make sure you have the following 3 **mandatory variables** ready:

```env
# 🔴 REQUIRED VARIABLES 🔴
MONGO_URL=your_mongodb_connection_string
SESSION_ID=your_session_id_here
PAIRING_NUMBER=your_whatsapp_number
```
> 💡 **NOTE:** To see the full list of advanced variables you can configure, check the **[Full Deployment Guide](https://cipher-md.vercel.app/deployment)**.

### 4️⃣ Deploy Bot
